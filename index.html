Создать группу реквизитов
▼

Группа = один получатель + один телефон. Все карты одного человека на одном телефоне — это одна группа.

💡1 получатель → 1 телефон → 1 группа реквизитов
  • ✓
    Раздел «Реквизиты» → подраздел «Группы»
  • ✓
    Нажать «Создать»
  • ✓
    Придумать название группы
  • ✓
    Поле «Телефон для CartelHouse» — оставить пустым
  • ✓
    Нажать «Сохранить»
Раздел Реквизиты → Группы
Форма «Создать новую группу»

Если тип оплаты — СБП: появится поле «Последние 4 цифры счёта (автоматика)»

ℹ️
• Если в SMS банка есть 4 цифры карты/счёта → вписать последние 4 цифры карты/счёта

Настройка SMS Forwarder
Приложение на телефоне, которое пересылает уведомления в систему.
📥
Актуальная версия APK:
https://sp-sms-forwarder.s3.eu-central-1.amazonaws.com/sp-sms-forwarder-0.15.0-16.apk
1✓
Разрешения и системные настройки телефона
▼
  • ✓
    Разрешить отправку и получение СМС (или пуш-уведомлений)
  • ✓
    Включить «Автозапуск» для SMS Forwarder
  • ✓
    Батарея → дать приложению доступ к энергии без ограничений
  • ✓
    Найти в настройках «Улучшенные уведомления» (или «Адаптивные») → выключить
⚠️«Адаптивные уведомления» скрывают содержимое — приложение не сможет прочитать текст.
2✓
Подключить приложение к площадке
▼
  • ✓
    Раздел «Реквизиты» → «Группы»
  • ✓
    Столбец «API ключ автоматизации» → нажать иконку QR
  • ✓
    Отсканировать QR внутри приложения
  • ✓
    Скопировать API ключ из столбца «API ключ автоматизации»
  • ✓
    В приложении вставить в поле Trader Token
  • ✓
    В поле Host вписать домен:
t.superfastapi.biz

Telegram-автоматизация
Используется если банк отправляет уведомления в Telegram-бот, или если пересылка настроена в личный бот.
1✓
Создать запись автоматизации
▼
👤Настройка выполняется через аккаунт с ролью Трейдер.
  • ✓
    Раздел «Реквизиты» → вкладка «Телеграм автоматизация»
  • ✓
    Нажать «Создать»
  • ✓
    Заполнить поле «Название»
  • ✓
    Вписать номер телефона Telegram-аккаунта
  • ✓
    Нажать «ОК»
Вкладка «Телеграм автоматизация» в разделе Реквизиты
Диалог «Создание телеграм автоматизации»
2✓
Войти в Telegram-аккаунт
▼
⚠️2FA на аккаунте должна быть выключена. Иначе вход не пройдёт.
  • ✓
    Нажать иконку карандаша напротив созданной автоматизации
  • ✓
    Нажать «Войти в аккаунт»
  • ✓
    Дождаться кода верификации в Telegram
  • ✓
    Ввести код и нажать «Сохранить»
Список автоматизаций — кнопка редактирования выделена красным
3✓
Указать бота и тип автозакрытия
▼
  • ✓
    В поле «Автоматизация» — вписать username Telegram-бота
📌Пример: бот банка Dushanbe City — @dc_next_tbot
